# Green Monkeys — Claude Code Project Guide

Persistent context for Claude Code. Read every session.

## What this is

A drinking-accountability app: pre-commitment videos before a session, drunk
videos during, regret videos the morning after, an honest self-verdict, and a
"Days since you were a(n) [configurable word]" streak. Mocking-but-affectionate
tone delivered by two characters (Green Monkeys, Captain Paranoia) with a
brutality dial. See `SPEC.md` — it is the source of truth for product behaviour.

## Tech stack (fixed)

- Swift 5.10+, SwiftUI, SwiftData (local only, no CloudKit)
- UNUserNotificationCenter with categories/actions; WatchConnectivity
- AVFoundation/UIImagePickerController for video capture, AVKit playback
- LocalAuthentication for the app lock
- Min iOS 17 / watchOS 10. No third-party dependencies without asking.

## Project generation & build

Generated by `xcodegen` from `project.yml` — never hand-edit the `.xcodeproj`
(it's gitignored). Same toolchain quirks as Doseify:

```bash
export DEVELOPER_DIR=/Applications/Xcode.app/Contents/Developer
xcodegen generate
xcodebuild -project GreenMonkeys.xcodeproj -scheme GreenMonkeys \
  -destination 'platform=iOS Simulator,name=iPhone 17' build
# tests
xcodebuild -project GreenMonkeys.xcodeproj -scheme GreenMonkeys \
  -destination 'platform=iOS Simulator,name=iPhone 17' test
```

## Layout

```
GreenMonkeys/            # iOS app: Models/ Views/ Services/
Shared/                  # DTOs compiled into both iOS and Watch targets
GreenMonkeysWatch/       # watchOS app (slim: streak + tonight's plan)
GreenMonkeysTests/       # Swift Testing (@Test / #expect)
```

## Conventions

Same as Doseify: 4-space indent, one primary type per file, Views/*View,
Services/*Service, `// MARK: -`, Swift Concurrency (no DispatchQueue), no `!`
force-unwraps outside tests. Every Service with logic gets tests; keep logic
testable as pure functions on value types where possible (see PatternService,
StreakService).

## Hard rules

1. **No analytics, telemetry, or any SDK that calls home. Local-only, always.**
2. **Videos never leave the device.** No share sheet, no export, no iCloud
   backup inclusion. Stored with complete file protection.
3. **Never delete or overwrite a recorded video or verdict except by explicit
   user action.**
4. **The verdict is self-reported.** The app may surface history and patterns
   but never auto-declares the user an idiot.
5. **Morning-after flow must always end forward-looking** (the "one change"
   step). Brutality dial scales the jokes, not the ending.
6. **No medical claims.** This is not a treatment for alcohol dependence; keep
   the NHS support link in Settings intact.

## Out of scope for v1

Cloud/sync/accounts, drink logging/BAC, widgets/complications, HealthKit,
AI-generated lines, sharing of any kind.

## User preferences

Owner prefers concise output and honest uncertainty. Avoid filler.
